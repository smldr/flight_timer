# Flight Timer

A relaxing, projector-optimised exam/prac timer. A plane takes off, arcs across the sky, and lands exactly when time runs out.

## How to run

Double-click `index.html`. That's it — no installation, no internet, no server.

Works in any modern browser (Chrome, Firefox, Safari, Edge).

## Usage

1. Enter a duration using the **hr / min / sec** inputs, or tap a preset (10 min → 3 hr)
2. Click **Begin Flight**
3. The plane takes off from the left. At the halfway point it's at the top of its arc. It descends and lands as time runs out.
4. When time's up, **"Time's Up / Pens Down"** appears. Click **↩ New Flight** to reset.

## Tips for the projector

- Run the browser **fullscreen** (F11 on Windows/Linux, Cmd+Ctrl+F on Mac) for the best look
- The top-left corner shows the landing time (e.g. "Landing · 2:45 pm") so students always know the end time
- The top-right corner shows the current time
- Dark background is easy on the projector bulb and readable from the back of the room
